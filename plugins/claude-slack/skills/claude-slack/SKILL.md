# claude-slack

引数: $ARGUMENTS

## 引数なし

使い方を表示:
```
/claude-slack on     - Slack routing を有効化
/claude-slack off    - Slack routing を無効化
/claude-slack config - Slack 接続設定
/claude-slack status - 現在の状態を表示
```

## 引数あり

引数に応じて対応する claude-slack エージェントを起動してください。
